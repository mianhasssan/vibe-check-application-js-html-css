## 🎨 Quick Customization

### Change Colors (2 minutes)
Edit `vibe-check-machine/index.css` lines 7-15:
```css
:root {
    --primary-color: #667eea;      /* Your color */
    --secondary-color: #764ba2;    /* Your color */
}
```

### Add Questions (5 minutes)
Edit `vibe-check-machine/index.js` questions array:
```javascript
const questions = [
    {
        text: "Your question?",
        choices: [
            { label: "Option 1", points: 0 },
            { label: "Option 2", points: 1 },
            // ... more options
        ]
    }
];
```

### Change Results (3 minutes)
Edit `computeVibe()` function in `index.js`:
```javascript
function computeVibe(totalScore) {
    if (totalScore <= 3) {
        return {
            emoji: "🛋️",
            title: "Custom Title",
            desc: "Custom description"
        };
    }
}
```
